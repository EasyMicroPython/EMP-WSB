# EMP WSB
该项目是 EMP-IDE 串口连接的转发工具，用于将串口的数据向浏览器进行实时的转发。


## Quick Start

### Install

pip install emp-wsb


### Usage

# wsb run [device] [port]
# devive: serial device name
# port: the port that websocket will listen to, default is 9000
# for example:

wsb run com3

wsb plugin running at:
    - local: http://localhost:9000
